# Hello World in Dart

A minimal Dart implementation of "Hello, World!".

## Installation

### macOS
```bash
brew install dart
```

### Linux
```bash
sudo apt install dart  # Ubuntu/Debian
sudo dnf install dart  # Fedora/RHEL
```

### Windows
Download from https://dart.dev/get-dart

## Running

```bash
dart main.dart
```

## Code Explanation

Uses print function for console output.
